# Летняя практика 
В рамках учебной ознакомительной практики выполнялось индивидуальное задание по теме «Процессы обработки документов на фармацевтическом предприятии». Практика включала последовательную работу на трёх платформах: SILA Union, BPMSoft и Visiology. <p>
**Моделирование предметной области на платформе [SILA Union](https://github.com/ed-dev-rd/Data_Analytics_Portfolio/blob/main/SILA%20Union.md)** <p>
На платформе SILA Union были построены четыре модели: организационная структура предприятия, дерево процессов в нотации VAD, EPC-модель стадии контроля качества и исполняемая BPMN-модель процесса обработки документов. Модели приведены в непротиворечивое состояние посредством перекрёстной проверки элементов и связей между ними.<p>
**Реализация бизнес-процесса на платформе [BRMSoft](https://github.com/ed-dev-rd/Data_Analytics_Portfolio/blob/main/BRMSoft.md)** <p>
На платформе BPMSoft спроектирован объект данных «Документ фармкомпании», создан соответствующий раздел, настроены справочники и организационные роли, разработана схема исполняемого бизнес-процесса, включающая двенадцать шагов. Процесс запущен и протестирован; работоспособность всех элементов подтверждена журналом процессов.<p>
**Аналитическая визуализация показателей на платформе [Visiology](https://github.com/ed-dev-rd/Data_Analytics_Portfolio/blob/main/Visiology.md)** <p>
На платформе Visiology подготовлен денормализованный набор данных из ста записей, отражающих обработку документов за год. На его основе построена аналитическая панель, содержащая ключевые показатели: долю отклонений, средний цикл обработки, стоимость обработки и распределение трудозатрат по стадиям процесса.<p>
